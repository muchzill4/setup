# Project Setup

## New Projects

When starting a project from scratch, run the appropriate `mise preset:*` task to bootstrap tooling and configuration:

| Preset | Command |
|---|---|
| Elixir | `mise preset:elixir` |
| Go | `mise preset:go` |
| Lua | `mise preset:lua` |
| Python | `mise preset:python` |
| Ruby | `mise preset:ruby` |

This is a required first step before writing any code.
